# User Personas and Journeys

## Personas

### P1 — First-Time Driver (primary)
Age 16–25. Preparing for first driver's license theory exam. Studies on phone in short sessions. Wants to know exactly what to study next and whether they are ready. Low tolerance for clutter.

### P2 — Retaker
Failed the theory exam at least once. Anxious. Needs weak-topic identification and realistic mock exams to build confidence. Reviews incorrect answers heavily.

### P3 — Adult Relocator
Age 30–55. Licensed elsewhere, must pass a local exam after moving. Time-poor; studies in commute gaps. Values search and bookmarks to target unfamiliar local rules.

### P4 — Administrator
Non-developer content manager. Creates and maintains exams, questions, images. Needs a simple web interface and confidence that edits appear correctly in the app.

## Journeys

### J1 — First study session (P1)
1. Install app, register with email.
2. Select country and driver's license category.
3. Land on dashboard (empty state points to Practice).
4. Start practice, all topics. Answer questions with immediate feedback and explanations.
5. Bookmark two confusing questions. Quit mid-session; progress saved.

### J2 — Targeted improvement (P2)
1. Login. Dashboard shows weak topics (e.g., "Right of way — 54%").
2. Start practice filtered to weak topic.
3. Afterwards, run review session of all previously incorrect questions.
4. Accuracy on topic rises; dashboard reflects it.

### J3 — Readiness check (P1/P2)
1. Start mock exam. Timer runs; questions randomized per real exam composition.
2. Submit (or auto-submit at timeout). See score, pass/fail, per-question review with explanations.
3. Dashboard pass-rate trend informs decision to book real exam.

### J4 — Quick lookup (P3)
1. Search "roundabout". Open matching question, read explanation, bookmark it.

### J5 — Content update (P4)
1. Login to admin panel. Edit a question's explanation, replace its image.
2. Change is live in app on next question fetch. Analytics shows the question's accuracy.
